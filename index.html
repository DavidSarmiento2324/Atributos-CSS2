Que Es CSS?

Una hoja de estilo en cascada, comúnmente conocida como CSS, es una capa de diseño sobre elementos HTML

  • Su Importancia En El Desarrollo Web:

    • CSS es una capa de diseño que aplica estilo a los documentos HTML, haciendo que las páginas web sean más presentables y controladas visualmente


  • Sus Principales Beneficios:

    • CSS es la mayor flexibilidad de diseño e interactividad que aporta al desarrollo web. Los desarrolladores tienen mayor control sobre el diseño, lo que les permite realizar cambios precisos por sección.

Que Es JavaScript?

JavaScript (JS) es un lenguaje de programación usado en el Front End y Back End de sitios web y apps móviles. Es un lenguaje interpretado que se ejecuta en el navegador, permitiendo crear contenido dinámico como menús, formularios, galerías y gráficos animados.

  • Su Importancia En El Desarrollo Web:

    • Interacción con el usuario: Por ejemplo, muestra ventanas emergentes con un campo para ingresar el correo u otros datos, y notificaciones sobre el uso de cookies.


    • Animación: Permite crear eventos que ocurren cuando el usuario presiona los botones, agregar efectos dinámicos de estilo, elementos animados, multimedia y mucho más.


    • Procesamiento de datos HTML: JS ayuda a completar campos obligatorios en formularios, verificar valores literales, numéricos y manejar elementos HTML en la página web.


    • Operaciones matemáticas: Los scripts de JS permiten agregar calculadoras en los sitios web, por ejemplo, en el carrito de compras de una tienda en línea.
Nombre del atributo Descripción Valores posibles Ejemplo de uso Imagen representativa Fuente de información
font-size Define el tamaño de la fuente del texto en una página web. Los valores (píxeles), (puntos), (centímetros), (milímetros). font-size: 16px; /* Tamaño en píxeles */ font-size: 1.5em; /* Tamaño relativo al elemento padre */
border-radius Define el radio de los bordes de un elemento, permitiendo (píxeles): Define el radio en unidades absolutas. Define el radio en proporción al tamaño del elemento- Toma el valor del elemento padre. border-radius: 10px; /* Bordes redondeados */ border-radius: 50%; /* Forma completamente circular */ Link De La Fuente
display Controla cómo se comportan los elementos en la página, definiendo si son bloques block, inline, flex, grid, none display: flex; /* Activa el modelo flexbox */ display: grid; /* Activa el modelo de cuadrícula */
flexbox (display: flex;) Un modelo de diseño que organiza elementos dentro de un contenedor flexible. Ideal para alineaciones y distribución de espacio dinámico entre elementos. flex-direction, row, column, row-reverse, column-reverse justify-content: center; /* Alinea al centro horizontalmente */ align-items: flex-start; /* Alinea al inicio verticalmente */
position Determina la ubicación de un elemento en la página. Static, relative, absolute, fixed, sticky position: absolute; /* Se posiciona relativo al contenedor padre */ position: sticky; /* Mantiene su posición mientras se hace scroll */
box-shadow Aplica sombras a un elemento. Incluye propiedades como desplazamiento, desenfoque y color para crear efectos visuales. desplazamiento horizontal y vertical, desenfoque, extensión, color box-shadow: 2px 4px 8px rgba(0, 0, 0, 0.2); /* Sombra con desenfoque */ box-shadow: none; /* Sin sombra */
overflow Controla cómo se maneja el contenido desbordado de un contenedor. visible, hidden, scroll, auto overflow: hidden; /* Oculta contenido desbordado */ overflow: scroll; /* Agrega barras de desplazamiento */
z-index Define el orden de apilamiento de elementos en la página. Los valores más altos se muestran por encima de los más bajos. valores enteros, auto z-index: 10; /* Apila este elemento sobre otros */ z-index: -1; /* Mueve el elemento detrás de otros */
opacity Ajusta la transparencia de un elemento. valores numericos entre 0 y 1 opacity: 0.5; /* 50% de transparencia */ opacity: 1; /* Totalmente opaco */
margin y padding Margin: Espacio externo entre un elemento y otros. Padding: Espacio interno entre el contenido de un elemento y sus bordes. Unidades absolutas, unidades relativas, valores especificos margin: 20px; /* Espacio externo alrededor del elemento */ padding: 15px; /* Espacio interno dentro del elemento */
